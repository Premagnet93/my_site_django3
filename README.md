# Доска объявлений

Веб‑приложение для размещения и просмотра объявлений с рубрикацией. Написано на Django.

## Функциональность

* Просмотр списка объявлений.
* Добавление новых объявлений.
* Фильтрация объявлений по рубрикам.
* Навигация по сайту (меню с ссылками на главную, добавление и рубрики).
* Адаптивный дизайн (корректное отображение на мобильных устройствах).

## Технологии

* **Python 3.8+**
* **Django 5.1**
* **SQLite** (база данных по умолчанию)
* **HTML5, CSS3** (вёрстка и стили)
* **Bootstrap 5** (опционально, для ускорения вёрстки)
